Margins are used to create space around elements, outside of any defined borders.

The margin property is a shorthand property for the following individual margin properties:

  • margin-top
  • margin-right
  • margin-bottom
  • margin-left
    • Margin Example 1
    
  <!DOCTYPE html>
  <html>
  <head>
  <style>
  div {
    border: 1px solid black;
    margin: 25px 50px 75px;
    background-color: lightblue;
  }
  </style>
  </head>
  <body>
  <h2>The margin shorthand property - 3 values</h2>

  <div>This div element has a top margin of 25px, a right and left margin of 50px, and a bottom margin of 75px.</div>

  </body>
  </html>
    Margin Example 2
    
  <!DOCTYPE html>
  <html>
  <head>
  <style>
  div {
    border: 1px solid black;
    margin-top: 50px;
    background-color: lightcoral;
  }
  </style>
  </head>
  <body>
  
  <h2>Using shorthand margin property</h2>
  
  <div>This div element has a margin of 50px on all sides.</div>
  
  </body>
  </html>
    Margin Example 3
    
  <!DOCTYPE html>
  <html>
  <head>
  <style>
  div {
    border: 2px solid black;
    margin-left: 50px;
    background-color: lightgreen;
  }
  </style>
  </head>
  <body>
  <h2>Two-value margin shorthand</h2>

  <div>This div element has a top and bottom margin of 10px, and left and right margins of 20px.</div>

  </body>
  </html>
    Margin Example 4
    
  <!DOCTYPE html>
  <html>
  <head>
  <style>
  div {
    border: 1px solid black;

    background-color: lightpink;

  </head>
  <body>
  <h2>Centering with margin</h2>

  <div>This div element is horizontally centered using margin: 0 auto.</div>

  </body>
  </html>
    Margin Example 5
    
  <!DOCTYPE html>
  <html>
  <head>
  <style>
  div {
    border: 1px solid black;
    margin-bottom:50px ;
    background-color: lightyellow;
  }
  </style>
  </head>
  <body>
  <h2>Percentage-based margins</h2>

  <div>This div element has top and bottom margins of 15% of the parent height and left and right margins of 10% of the parent width.</div>

  </body>

    The CSS border properties allow you to specify the style, width, and color of an element's border.

    The border-style property specifies what kind of border to display.
    The following values are allowed:

  • dotted - Defines a dotted border
  • dashed - Defines a dashed border
  • solid - Defines a solid border
  • double - Defines a double border

  • groove - Defines a 3D grooved border. The effect depends on the border-color value

  • ridge - Defines a 3D ridged border. The effect depends on the border-color value

  • inset - Defines a 3D inset border. The effect depends on the border-color valuet

  • outset - Defines a 3D outset border. The effect depends on the border-color value

  • none - Defines no border

  • Padding-left

    • Border
    
  <!DOCTYPE html>
  <html>
  <head>
  <style>
  p.dotted {border-style: dotted;}
  p.dashed {border-style: dashed;}
  p.solid {border-style: solid;}
  p.double {border-style: double;}
  p.groove {border-style: groove;}
  p.ridge {border-style: ridge;}
  p.inset {border-style: inset;}
  p.outset {border-style: outset;}
  p.none {border-style: none;}
  p.hidden {border-style: hidden;}
  p.mix {border-style: dotted dashed solid double;}
  </style>
  </head>
  <body>
  
  <h2>The border-style Property</h2>
  <p>This property specifies what kind of border to display:</p>

  <p class="solid">A solid border.</p>
  <p class="double">A double border.</p>
  <p class="groove">A groove border.</p>
  <p class="ridge">A ridge border.</p>
  <p class="inset">An inset border.</p>

  </body>
